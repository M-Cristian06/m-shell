# m-shell

## a shell created in python 

### Commands:

- ` info>>version ` - Shows the current shell version
- ` info>>help ` - Displays all shell commands
- ` rq>>[url] ` - request url to test websites returned status code or your own internet connection


### instalation

1. Clone repo:
    ```bash 
    git clone https://github.com/M-Cristian06/m-shell.git
    cd m-shell
    ```

2. Create env:
    ```bash 
    python3 -m venv env
    ```

3. Activate env:
    ```bash 
    source env/bin/activate
    ```

4. Install requirements: 
    ```bash 
    pip install -r requirements.txt
    ``` 

5. Run:
    ```bash
    python3 main.py
    ```
